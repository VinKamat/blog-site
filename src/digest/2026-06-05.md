### [Anthropic says AI is now speeding up AI development inside the company](https://www.anthropic.com/institute/recursive-self-improvement)
Anthropic says Claude authored more than 80% of the code merged into its codebase as of May 2026, and the typical engineer is shipping 8x more code than in 2024. The company frames this as early evidence that AI is accelerating AI development, even though humans still set goals and make higher-level judgments.
> This is the kind of signal enterprise teams should pay attention to. The real story is not autocomplete. It is longer-running agents taking on meaningful engineering work, which raises the bar for review, evals, and governance.

### [Microsoft introduced MAI-Thinking-1 and MAI-Code-1-Flash](https://simonwillison.net/2026/Jun/2/microsofts-new-models/)
Microsoft announced MAI-Thinking-1, a reasoning model with 1T total parameters and 35B active parameters for select early partners, plus MAI-Code-1-Flash, a 137B model with 5B active parameters rolling out to GitHub Copilot users in VS Code. Simon Willison also highlighted Microsoft's claim that both models were built end-to-end with commercially licensed data, then noted the technical paper still describes large-scale web crawl training.
> Microsoft keeps tightening the loop between model, developer tool, and distribution channel. That matters more than leaderboard chest-thumping because it gives enterprises a cleaner path from experiment to standard tooling.
</div>

### [NVIDIA released Nemotron 3 Ultra and Nemotron 3.5 ASR](https://www.latent.space/p/ainews-not-much-happened-today-7a8)
Latent Space's June 5 roundup says NVIDIA launched Nemotron 3 Ultra, an open 550B MoE model with 55B active parameters and 1M context built for long-running agent workloads, alongside Nemotron 3.5 ASR for low-latency streaming speech. The same roundup says the release landed quickly across serving stacks including vLLM, Modal, Together, Fireworks, Baseten, and Ollama cloud.
> Open model releases matter when the deployment story is ready on day one. This is the shape of competition now: model quality, serving speed, context length, and ecosystem support all shipping together.
</div>

### [ChatGPT memory keeps expanding as usage crosses a new scale milestone](https://www.latent.space/p/ainews-not-much-happened-today-7a8)
Latent Space's June 5 roundup says ChatGPT crossed 1 billion monthly active users and OpenAI rolled out stronger memory features, including summaries, more controls, and larger memory capacity for paid users in the US. OpenAI's memory update page says memory now draws from saved memories and past conversations, with user controls to disable or manage both.
> Persistent memory is becoming core product infrastructure. Once chat systems can carry context across sessions at this scale, the product starts behaving less like a stateless tool and more like a working environment.
</div>

### [Uber capped monthly AI coding spend per tool](https://simonwillison.net/2026/Jun/3/uber-caps-usage/)
According to Simon Willison's June 3 note on Bloomberg's reporting, Uber is limiting employees to $1,500 per month for each AI coding tool such as Claude Code or Cursor. The cap appears to follow a period of rapid AI budget burn and gives a concrete view into how large enterprises are starting to price agentic coding tools internally.
> This is the FinOps phase of enterprise AI. Interest is still high. Procurement and platform teams now need hard usage guardrails before these tools become part of normal engineering spend.
</div>

### [Apple approved Poke as the first AI agent on Messages for Business](https://techcrunch.com/2026/06/04/apple-approves-poke-as-the-first-ai-agent-on-its-messages-for-business-platform/)
TechCrunch reports that Poke became the first standalone AI agent approved to run on Apple's Messages for Business platform, giving it access to iMessage as a distribution channel. The startup says Apple charges it on a per-user basis, which creates a new platform toll for AI agent companies building on messaging rails.
> Messaging is becoming a serious interface for agents. The important detail here is distribution economics: once platform owners can charge per active user, agent margins and go-to-market strategy change fast.
</div>

### [Meta is building AI data centers in tents to speed deployment](https://techcrunch.com/2026/06/04/meta-steals-a-tactic-from-tesla-and-builds-data-centers-in-tents/)
TechCrunch reports that Meta has built rapid deployment tent structures in New Albany, Ohio as part of its push to shorten AI data center construction timelines. The report says the site is paired with 200 megawatts of modular gas turbines and reflects Meta's broader plan to spend heavily on AI infrastructure.
> The AI race is getting very physical. When companies start wrapping billions of dollars of chips in temporary structures, you can see how much schedule pressure is driving infrastructure decisions.
