### [Release: datasette-agent-edit 0.1a0](https://simonwillison.net/2026/Jun/7/datasette-agent-edit/)
Simon Willison released `datasette-agent-edit`, a base plugin for Datasette Agent that standardizes file editing around `view`, `str_replace`, and `insert` operations. The goal is to make agent-driven text edits more reliable across Markdown, SQL, SVG, and similar artifacts.
> This is the kind of primitive that matters. Teams building agents into enterprise workflows need editing operations that are predictable, bounded, and easy to audit.
</div>

<div class="digest-entry">
### [Running Python code in a sandbox with MicroPython and WASM](https://simonwillison.net/2026/Jun/6/micropython-in-a-sandbox/)
Willison also detailed `micropython-wasm`, an alpha package that runs Python inside WebAssembly using `wasmtime`, with memory limits and controlled host functions. He is already using it in a Datasette Agent plugin to execute code with tighter isolation than normal Python plugins.
> This hits a real pain point for agent systems. If code-executing agents are going to show up in production, sandbox design needs to move from hand-wavy to concrete fast.
</div>

### [How to Stop Shipping Low-Quality RL Environments (with Examples)](https://www.latent.space/p/bad-envs)
Auriel Wright argues that broken RL environments poison training data and teach models the wrong behavior through stale state, bad rewards, and flaky resets. The piece frames environment quality as a software engineering problem with direct impact on model performance.
> Good agents depend on good harnesses. If the training environment lies, the model learns lies, and that debt shows up later as brittle production behavior.
</div>

### [OpenAI unveils Lockdown Mode to protect sensitive data from prompt injection attacks](https://techcrunch.com/2026/06/06/openai-unveils-lockdown-mode-to-protect-sensitive-data-from-prompt-injection-attacks/)
OpenAI said Lockdown Mode reduces prompt-injection risk by disabling live web browsing, web images, deep research, and agent mode, while still allowing cached content. The company says the feature is aimed at people and organizations handling sensitive data and is rolling out to self-serve ChatGPT Business and some eligible personal accounts.
> This is a clean signal that prompt injection is now a mainstream product concern. Enterprise rollouts need security modes like this, plus policy controls and clear user education.
</div>

### [Google will pay SpaceX $920M per month for compute](https://techcrunch.com/2026/06/05/google-will-pay-spacex-920m-per-month-for-compute/)
TechCrunch reports Google agreed to pay SpaceX $920 million per month from October 2026 through June 2029 for access to roughly 110,000 NVIDIA GPUs, CPUs, memory, and related components. Google described the deal as bridge capacity to meet stronger-than-expected demand for its agent platform and Gemini Enterprise offerings.
> AI infrastructure demand is still outrunning owned capacity, even for hyperscalers. When that happens, architecture decisions start getting shaped by supply contracts as much as by product strategy.
</div>

### [The token bill comes due: Inside the industry scramble to manage AI’s runaway costs](https://techcrunch.com/2026/06/05/the-token-bill-comes-due-inside-the-industry-scramble-to-manage-ais-runaway-costs/)
TechCrunch says enterprises are overrunning AI budgets as agentic usage drives token consumption up, even while per-token prices fall. The Linux Foundation is launching the Tokenomics Foundation to standardize how organizations measure, compare, and control token spend.
> This feels like the FinOps phase of enterprise AI finally arriving. Once budgets tighten, every agent flow has to justify its cost with hard telemetry and real business value.
