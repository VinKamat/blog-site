### [Introducing Mistral Small 4](https://simonwillison.net/2026/Mar/16/mistral-small-4/)
Mistral released Mistral Small 4, an Apache 2 licensed 119B Mixture-of-Experts model with 6B active parameters. It combines reasoning, multimodal, and agentic coding capabilities in one model and exposes a `reasoning_effort` setting with none and high modes.
> Open-weight multimodal models with reasoning are getting more practical. This matters because teams can cut down model sprawl and evaluate one stack for several workloads. The missing API detail around reasoning control is still a real integration gap.

### [Use subagents and custom agents in Codex](https://simonwillison.net/2026/Mar/16/codex-subagents/)
OpenAI moved Codex subagents into general availability and added support for custom agents defined as TOML files in `~/.codex/agents/`. The default roles include explorer, worker, and default, and teams can bind custom instructions and model choices to named agents.
> This pushes coding agents closer to a repeatable operating model. Once roles, prompts, and model selection live in config, agent orchestration starts to look like software architecture. Teams will use this to standardize workflows, not just speed up one-off tasks.
</div>

### [Nvidia’s version of OpenClaw could solve its biggest problem: security](https://techcrunch.com/2026/03/16/nvidias-version-of-openclaw-could-solve-its-biggest-problem-security/)
Nvidia announced NemoClaw at GTC, an open-source enterprise AI agent platform built on top of OpenClaw with stronger security and privacy controls. Nvidia says it is hardware agnostic, integrates with NeMo, and is starting in alpha with production sandboxing still in progress.
> This is the enterprise adoption pattern in plain view. Open ecosystems create the demand, then governance, sandboxing, and policy control decide who gets deployed. Security is where agent platforms stop being a toy and start becoming infrastructure.

### [[AINews] NVIDIA GTC: Jensen goes hard on OpenClaw, Vera CPU, and announces $1T sales backlog in 2027](https://www.latent.space/p/ainews-nvidia-gtc-jensen-goes-hard)
Latent Space highlighted Moonshot’s Attention Residuals paper as one of the strongest technical stories of the cycle. The paper replaces fixed residual accumulation with input-dependent attention over prior layers and claims a 1.25x compute advantage with less than 2% inference latency overhead on Kimi Linear 48B.
> This is the kind of architecture work that deserves attention. If the gains hold outside the paper, deep models may get better memory behavior and better efficiency at the same time. People building long-context or multi-stage systems should watch this closely.

### [Memories AI is building the visual memory layer for wearables and robotics](https://techcrunch.com/2026/03/16/memories-ai-is-building-the-visual-memory-layer-for-wearables-and-robotics/)
Memories.ai is building infrastructure for wearables and robots to store and retrieve visual memories, using Nvidia Cosmos-Reason 2 and Metropolis in a new GTC partnership. The startup says it has raised $16 million and is focused on indexing video data for later recall through its large visual memory model.
> Physical AI needs durable multimodal memory. A robot or wearable that cannot retrieve what it has seen stays shallow and stateless. If this layer matures, it opens up better assistants, better embodied agents, and stronger retrieval design beyond text.
</div>

### [The dictionary sues OpenAI](https://techcrunch.com/2026/03/16/merriam-webster-openai-encyclopedia-brittanica-lawsuit/)
Encyclopedia Britannica and Merriam-Webster sued OpenAI, alleging that nearly 100,000 copyrighted articles were scraped for model training and reused in outputs and retrieval workflows without permission. The suit also claims hallucinated answers falsely attributed to Britannica create trademark and trust issues.
> Legal pressure is moving toward retrieval behavior, attribution, and output provenance. Enterprise teams should treat content licensing as a design constraint now. This will shape vendor selection, grounding strategy, and audit requirements.
</div>

### [Picsart now allows creators to ‘hire’ AI assistants through agent marketplace](https://techcrunch.com/2026/03/16/picsart-now-allows-creators-to-hire-ai-assistants-through-agent-marketplace/)
Picsart launched an AI agent marketplace with agents for Shopify product work, social resizing, style remixing, and bulk background changes. The platform includes autonomy levels and messaging integrations so users can approve or supervise agent actions.
> This is where agent UX gets real. Narrow agents with clear tools, bounded scope, and approval checkpoints are easier to trust and easier to ship. Expect more vertical marketplaces like this before general-purpose consumer agents become dependable.
