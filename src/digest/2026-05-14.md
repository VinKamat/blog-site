### [Brain File Format for AI Agents — One File, Any LLM, Sub-ms Queries](https://news.ycombinator.com/item?id=47068547)

AgenticMemory is a Rust-based binary graph format that stores an agent's full knowledge graph as typed nodes and edges (caused_by, supports, supersedes). Performance: 276ns to add a node, 9ms similarity search across 100K entries, ~24MB for a year of daily use. Works with any LLM, zero dependencies.

> Agent memory is still largely unsolved in production deployments. Vector DBs and markdown dumps lose causal structure — they can't represent that one fact superseded another, or that a decision caused a downstream inference. A typed graph that tracks these relationships is architecturally closer to what reasoning agents actually need. Early-stage, but the framing of the problem is right.

</div>
