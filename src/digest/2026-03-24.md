### [AgentArmor: Open-Source 8-Layer Security Framework for AI Agents](https://github.com/Agastya910/agentarmor)

AgentArmor wraps agentic architectures with 8 independent security layers covering prompt injection detection, storage encryption, context separation, action risk scoring, network egress control, PII redaction, inter-agent authentication, and JIT identity — tested against the December 2025 OWASP ASI spec.

> Most production agent deployments have essentially no security architecture beyond trusting the LLM. This maps neatly to the OWASP ASI risks and gives enterprise teams a concrete checklist. The action risk scoring layer (READ=1, EXECUTE=8, ADMIN=10) is the pattern that should be standard in any enterprise agentic deployment.
