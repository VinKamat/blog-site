### [Rewriting Bun in Rust](https://simonwillison.net/2026/Jul/8/rewriting-bun-in-rust/)
Simon Willison highlighted Jarred Sumner's write-up on Bun's Zig-to-Rust rewrite, where an agent harness used the TypeScript test suite as a conformance system and drove much of the port. The post says the Rust implementation has already been live in Claude Code since June 17, with startup around 10% faster on Linux and an estimated API-price token cost of about $165,000 for the pre-merge effort.
> This is one of the clearest agentic engineering case studies yet. The interesting part is not just code generation, it is the harness discipline around tests, adversarial review, and fixing the workflow instead of patching every output by hand.

### [Introducing GPT-Live](https://simonwillison.net/2026/Jul/8/introducing-gptlive/)
Willison surfaced OpenAI's new GPT-Live voice mode, which can keep a spoken conversation going while delegating harder tasks such as web search or deeper reasoning to GPT-5.5 in the background. The release updates ChatGPT voice from an older GPT-4o-era model to a newer live system aimed at more natural long-form interaction.
> Voice is starting to look like a routing surface, not a standalone model experience. The architecture question shifts toward when the system hands work to a heavier model, how state carries across, and how much latency users will tolerate in exchange for depth.
</div>

### [SpaceXAI releases Grok 4.5, which Elon describes as an 'Opus-class model'](https://techcrunch.com/2026/07/08/spacexai-releases-grok-4-5-which-elon-describes-as-an-opus-class-model/)

Latent Space's same-day roundup adds that the model was trained with Cursor, launched into Grok Build, the API, and Cursor, and was framed around strong coding and agent workflows.
