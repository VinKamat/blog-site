### [OpenAI Acquires Promptfoo to Secure Its AI Agents](https://techcrunch.com/2026/03/09/openai-acquires-promptfoo-to-secure-its-ai-agents/)

OpenAI acquired Promptfoo, the open-source LLM security and red-teaming toolkit, to harden its agents against prompt injection, misuse, and adversarial inputs in enterprise deployments.

> Frontier labs are now buying security capabilities rather than building them — which tells you how fast the attack surface is growing. For architects deploying agentic pipelines, prompt injection and misuse at scale are no longer theoretical. This acquisition sets the bar for what "production-ready" means.

### [Anthropic Launches Code Review Tool for AI-Generated Code](https://techcrunch.com/2026/03/09/anthropic-launches-code-review-tool-to-check-flood-of-ai-generated-code/)

Anthropic launched Code Review inside Claude Code — a multi-agent system that automatically analyzes AI-generated code, flags logic errors, and helps enterprise dev teams manage the growing volume of vibe-coded output.

> The review bottleneck was always going to appear once coding agents became mainstream. A dedicated review agent working alongside the coding agent is the right architecture. The open question is whether it catches security vulnerabilities, not just logic bugs — that's the enterprise gating criteria.

### [NVIDIA Dynamo: Agent Inference at Planetary Scale](https://www.latent.space/p/nvidia-brev-dynamo)

In a pre-GTC Latent Space episode, NVIDIA's Kyle Kranen walks through Dynamo — a datacenter-scale inference framework for LLM agents using prefill/decode disaggregation, Kubernetes orchestration, and cost/latency/quality tradeoff controls across SGLang, TRT-LLM, and vLLM.

> Enterprise agentic workloads aren't just an API call problem — they're an infrastructure problem. Dynamo is NVIDIA's bet that inference for agents at scale requires purpose-built orchestration, not generic cloud load balancing. The disaggregation of prefill vs. decode is particularly relevant for long-running agents with big context windows.

### [Anthropic Sues Defense Department Over Supply-Chain Risk Designation](https://techcrunch.com/2026/03/09/anthropic-sues-defense-department-over-supply-chain-risk-designation/)

Anthropic filed suit against the DOD after the agency labeled it a national supply-chain risk, calling the designation "unprecedented and unlawful." Over 30 OpenAI and Google DeepMind employees publicly sided with Anthropic. Claude's app is now outpacing ChatGPT in daily new installs in the aftermath.

> AI governance, use restrictions, and export control are now binding contract constraints — not just legal fine print. Any enterprise architect evaluating AI vendors for regulated industries needs to treat vendor policy positions as technical risk factors, not just PR noise.

### [Agentic Manual Testing](https://simonwillison.net/guides/agentic-engineering-patterns/agentic-manual-testing/)

Simon Willison's Agentic Engineering Patterns guide covers how coding agents can manually test their own output — using `python -c`, `curl`, Playwright, and browser automation — to catch issues that automated tests miss. Includes practical patterns for web UIs using Playwright and his Rodney/Showboat tools.

> This is the practical gap between "AI writes code" and "AI ships working code." Getting an agent to verify its own work through real execution, not just assertion, is what makes an autonomous contributor. The Playwright + screenshot + vision loop in particular is a pattern worth stealing for enterprise app testing.

### [Introducing GPT-5.4](https://openai.com/index/introducing-gpt-5-4/)

OpenAI released gpt-5.4 and gpt-5.4-pro with a 1 million token context window and August 2025 knowledge cutoff. GPT-5.4 outperforms the coding-specialist GPT-5.3-Codex on benchmarks and shows a 28% improvement on spreadsheet and document modeling tasks compared to GPT-5.2.

> The 1M context window is the headline but the spreadsheet modeling benchmark is the interesting one — that's a direct play at financial modeling and business intelligence workflows. Enterprise use cases that previously needed custom fine-tuning may now be within reach of prompt engineering alone.

</div>

### [Yann LeCun's AMI Labs Raises $1.03 Billion to Build World Models](https://techcrunch.com/2026/03/09/yann-lecuns-ami-labs-raises-1-03-billion-to-build-world-models/)

AMI Labs, LeCun's post-Meta venture, raised $1.03 billion at a $3.5 billion pre-money valuation to pursue world model AI — architectures that model physical causality and structure rather than token prediction.

> LeCun has argued for years that LLMs hit a fundamental ceiling on reasoning and planning. $1B gives him the compute to run that experiment at scale. If world models outperform on embodied or long-horizon planning tasks, the enterprise AI stack gets a lot more interesting — and more complex.

</div>
