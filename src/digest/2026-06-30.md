### [Ornith-1.0: Self-Scaffolding LLMs for Agentic Coding](https://simonwillison.net/2026/Jun/29/ornith/)
Simon Willison highlighted Ornith-1.0, a new MIT-licensed open-weights coding model from DeepReinforce. The release includes 9B and 31B dense variants plus 35B and 397B MoE variants, built on Gemma 4 and Qwen 3.5, with strong coding benchmark results for its class.
> Open coding models matter when they can actually hold up inside an agent loop. Ornith looks like one of the few new releases worth real evaluation for self-hosted coding workflows.

### [Meta's Brain2Qwerty v2 pushes non-invasive brain-to-text forward](https://www.latent.space/p/ainews-not-much-happened-today-07e)
Latent Space's June 30 AINews roundup says Meta unveiled Brain2Qwerty v2, a real-time sentence decoder trained on non-invasive brain signals. Meta is also releasing training code for v1 and v2, while BCBL is releasing the v1 dataset.
> This is still lab work, but the stack is getting serious. When language models start improving signal decoding pipelines, human-computer interfaces get a lot more interesting.

### [DeepSeek's DSpark sharpens speculative decoding](https://www.latent.space/p/ainews-not-much-happened-today-07e)
Latent Space reports that DeepSeek's DSpark improved accepted token length by 30.9% over Eagle3 and 16.3% over DFlash on Qwen3-4B. The same roundup says the vLLM community is already working on integration.
> Inference speed is still a systems problem. Gains like this matter because they hit cost, latency, and throughput at the same time.
</div>

### [Devin Fusion aims to cut coding-agent cost with hybrid routing](https://www.latent.space/p/ainews-not-much-happened-today-07e)
Latent Space says Cognition launched Devin Fusion, a hybrid-model coding harness that claims 35% lower cost while keeping Fable-level quality. The broader theme in the roundup is mid-session routing, where an expensive planner delegates bounded work to cheaper models.
> This is where agent architecture is headed. Model quality still matters, but harness design is starting to decide the unit economics.
</div>

### [Cursor now has a mobile app for guiding your coding agent on the go](https://techcrunch.com/2026/06/29/cursor-now-has-a-mobile-app-for-guiding-your-coding-agent-on-the-go/)
TechCrunch reports that Cursor launched an iOS app for remote oversight of coding agents. Latent Space adds that the release includes always-on cloud agents, live activities, and diff review from a phone.
> Mobile control will pull coding agents closer to normal team workflow. Once review and intervention work from anywhere, long-running agents become easier to trust operationally.
