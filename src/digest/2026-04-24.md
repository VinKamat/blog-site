### [DeepSeek V4 — almost on the frontier, a fraction of the price](https://simonwillison.net/2026/Apr/24/deepseek-v4/)

DeepSeek drops two new models: V4-Pro (1.6T total parameters, 49B active) and V4-Flash (284B total, 13B active), both with 1M token context windows. Flash prices at $0.14/M input — the cheapest available, beating even GPT-5.4 Nano. Pro at $1.74/M input undercuts every Western frontier model. Both are MIT licensed and available on Hugging Face now. The efficiency gains come from architectural improvements that cut FLOPs and KV cache size dramatically at long context lengths.

> The pricing is the real story. V4-Pro at $1.74/M input for frontier-competitive performance changes the enterprise cost calculus for high-volume agentic workloads. If you're running multi-step agent chains with long context, the math on DeepSeek just got very hard to ignore.

### [Schneier on LLM vulnerabilities, agentic AI, and "trusting trust"](https://herbsutter.com/2025/10/23/schneier-on-llm-vulnerabilities-agentic-ai-and-trusting-trust/)

Bruce Schneier applies Ken Thompson's classic "trusting trust" framework to LLM-based systems, examining how vulnerabilities compound when models take autonomous actions. The argument: you can't verify safety by auditing the model alone — you have to trust the entire stack the model touches.

> This is exactly the right lens for enterprise agentic architecture. When an agent chains tool calls, writes code, and hits external APIs, the attack surface is the whole execution graph. Security reviews that only inspect the model are missing the point. Governance frameworks need to catch up to this reality.
